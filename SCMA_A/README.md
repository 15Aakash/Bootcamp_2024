# SCMA_A – Supply Chain Management & Analytics

This folder contains assignments completed as part of the **Supply Chain Management & Analytics (SCMA)** module under the Bootcamp 2024 program.

## Folder Structure
- `A1` – Assignment 1  
  Covers foundational supply chain analytics concepts, implemented through multiple parts using Python and R.
  
- `A2` – Assignment 2  
  Focuses on more advanced analytical techniques and deeper interpretation of supply chain–related problems.

Each assignment folder includes:
- Jupyter Notebook files (`.ipynb`)
- R scripts (`.r`)
- Exported PDF files
- A dedicated README explaining the assignment contents

## Tools & Technologies
- Python
- R
- Pandas
- NumPy
- Jupyter Notebook
